# Projeto Flask + PostgreSQL

Aplicação web construída com **Flask** e **PostgreSQL**, com front-end simples em **HTML e CSS**.
O objetivo original é **facilitar e apoiar o trabalho de um amigo** em suas funções diárias.

---

## Tecnologias

* **Backend:** Python, Flask
* **Banco de dados:** PostgreSQL (AWS RDS)
* **Front-end:** HTML, CSS
* **Armazenamento em Nuvem** AWS S3
